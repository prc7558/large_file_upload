# Client Storage Folder

This represents the "local" hard drive of the user.

## Purpose
- This folder acts as the source for the file upload simulation.
- Any file you put here will be available for "uploading" to the simulated server.

## Files
- You can place any files here (text, images, videos) to test the simulation.
- Large files (1MB+) are recommended to better see the progress tracking and resume features.
