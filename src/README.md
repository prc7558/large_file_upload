# Source Code Directory

This directory contains the Java source files for the Local File Upload Simulator.

## Sub-packages
- **com.simulator**: Contains the entry point class `Main.java`.
- **com.simulator.core**: Contains the heart of the upload simulation and chunk processing logic.
- **com.simulator.util**: Contains utility classes for progress bars and UI formatting.

## How it contributes
The `src` folder is the brain of the project. It defines how data is read from the "client" and written to the "server", managing the lifecycle of the simulation process.
